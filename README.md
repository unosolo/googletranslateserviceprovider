## About Google Service Provider

- It uses a Service Provider approach so it can be change later to another translation service.
- The logic of the connection with Google Translation Service is missing because this only for testing purpose and my
 country is not allowed to access the Cloud Google API.
- It  simulates a simple translation and the Service Provider will always return a solo translation. No matter what
 text is provided to be translated.
